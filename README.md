# helmingway

## Features

- [ ] [設定](./helmingway.config.yaml)に基づいて、Helm チャートの alias ごとのレンダリング結果をプレビューできる
- [ ] プレビュー結果を UI 上でファイル単位に絞り込める
- [ ] プレビュー結果を UI 上で Kubernetes リソース単位に絞り込める
- [ ] Helm チャートのレンダリング結果について、alias 間の差分をチェックできる
- [ ] Helm チャートのレンダリング結果と、実際の Kubernetes リソースとの差分をチェックできる
- [ ] Helm チャートのレンダリング結果について、git commit 間の差分をチェックできる
